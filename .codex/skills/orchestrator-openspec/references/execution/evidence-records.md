# 证据记录模板

这份文档回答两个问题：

- 证据应该如何记录
- 什么证据足以支持当前轮收口

## 一、推荐的最小证据记录

- 规格入口：
- 任务包入口：
- 验证方式：
- 核心结果：
- 裁定：
- 轮次总结入口：

## 二、什么证据足以支持收口

满足全部条件时，通常可视为证据足够：

- 能指出规格或任务包来源
- 能指出验证方式或静态判断依据
- 能解释为什么接受、驳回或阻塞
- 能回到本轮总结

如果缺少其中任一项，主代理应谨慎收口。

## 三、证据判定短清单

主代理只需快速问四个问题：

1. 我能指出这次判断对应哪份规格或任务吗
2. 我能指出我依据了什么验证或静态证据吗
3. 我能解释为什么当前裁定成立吗
4. 我能让下一轮接上这次结论吗

四项里若有一项答不出来，证据通常还不够。
