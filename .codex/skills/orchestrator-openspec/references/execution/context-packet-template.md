# 上下文包模板

这份模板用于在长任务中一次性输出最小可续接上下文。

它默认同时指出三类记忆的落点：

- 规范记忆
- 执行记忆
- 经验记忆

## 标准骨架

### 1. 当前推进状态

- 当前轮唯一目标：
- 当前阶段：
- 当前完成情况：
- 当前阻塞项：
- 当前验证目标：
- 下一步：

### 2. 规范记忆落点

- 当前 change / 规格入口：
- 当前任务包来源：
- 当前保留约束：
- 当前规范记忆是否稳定：

### 3. 执行记忆落点

- 当前执行记忆包位置：
- 下一轮续接时最先看的信息：
- 当前是否需要阶段回退：

### 4. 经验记忆落点

- 本轮是否新增经验记忆：
- 新增经验摘要：
- 经验落点：
- 是否需要进入知识闭环：

## 使用原则

- 只保留下一轮继续推进所需信息
- 已关闭问题不再保留
- 下一步必须是单一动作
- 规范记忆优先回链到 OpenSpec 工件或冻结任务包
- 执行记忆只保留当前轮和下一轮续接必需内容
- 经验记忆只记录可复用的错误模式、修正和落点
- 如果当前阶段或目标说不清，应先回退而不是继续填写
